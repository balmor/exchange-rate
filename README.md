# Kalkulator różnic kursowych

Aplikacja oblicza różnice kursowe na podstawie podanej waluty

## Jak zacząć

Instalacja pakietów:

```
yarn install
```

Uruchomienie lokalnie w trybie development:

```
yarn start
```

Tworzenie aplikacji:

```
yarn build
```
